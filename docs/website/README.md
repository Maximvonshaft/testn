## Execution boundary

After the specification is approved, implementation must be split into independently reviewable workstreams:

1. Production Homepage Shell
2. Golden Wall Visual Construction Fix
3. Product Data and Compatibility
4. CRM / Lead Flow
5. CMS / Trust Evidence / Resources

These subsystems must not be bundled into one oversized pull request.
