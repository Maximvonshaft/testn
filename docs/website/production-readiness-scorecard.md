# 当前 Material Studio 原型生产就绪度审计 v0.6

**基线：** 用户测试的 Golden Wall v0.5.3 视觉与现有 Draft PR #16 的产品范围。  
**用途：** 决定哪些内容可保留、哪些必须重做、哪些不应进入生产官网。

## 评分说明

- 0–39：原型 / 缺失
- 40–59：方向成立但不可生产发布
- 60–79：可进入受控试点
- 80–100：生产级

## 维度评分

| 维度 | 分数 | 主要证据 | 结论 |
|---|---:|---|---|
| 品牌视觉 | 78 | 黑金视觉、空间主图、清晰控制面板 | 保留方向，修正文案和字号 |
| 商业定位 | 52 | 有 Wall Composer，但公司产品系统不清楚 | 需公司级定位与系统导航 |
| 信息架构 | 32 | 单页原型，导航多数无真实内容 | 阻断生产发布 |

**商业官网综合：43/100**  
**生产上线综合：30/100**

## Blocking Findings

1. 当前页面是互动原型，不是完整官网。
2. Golden Wall 白色大理石存在“高于墙体 / 浮在建筑前 / 发光屏”冲突。
3. 顶部、底部、左右没有可解释的建筑收口。
4. 导航指向不完整或空内容。
5. 产品代码、规格和兼容性不是正式主数据。
6. 客户界面出现内部测试术语。
7. 缺少工厂、质量、测试、项目和资料证据。
8. 询盘未进入真实商业系统。
9. 本地运行架构不具备生产发布条件。

## 必须保留

- 高端、克制的整体视觉方向
- Material / Profile / Finish 的选择逻辑
- Original / Configured 对比
- 单一黄金场景先通过，再扩展的策略
- 离线对位资产避免运行时坐标漂移的思路

## 必须删除或改写

- Scene-aligned demo
- Outside-wall invariant
- 0 modified pixels
- Verified by regression test
- Demo code
- 虚假或空导航
- “Specification only”这类不能帮助决策的标签

## Golden Wall 下一轮视觉验收

### 几何与构造

- 顶部边界与天花真实衔接，不超高。
- 底部有踢脚、暗缝、收边或自然落地阴影。
- 左右边界有厚度、退缝或压边。
- 不出现墙板浮在空间前的感觉。

### 材质与光照

- 降低白色材质的背光屏效果。
- 使用非镜像、低重复的大板纹理。
- 板缝和板宽符合真实规格。
- 保留环境光、阴影和局部反射。

### 客户体验

- 默认方案应当自然、可信，不优先展示最容易暴露缺陷的过亮白材质。
- 客户可见的是产品和项目价值，不是开发验证指标。

## Go / No-Go

### 可以进入下一阶段的条件

- 商业网站规格获得确认。
- Homepage Blueprint 获得确认。
- 产品主数据负责人提供第一批真实产品代码和规格。
- Golden Wall 建筑落位修复方案通过视觉评审。

### 当前结论

**No-Go for production launch.**  
**Go for structured specification and a new production homepage shell after approval.**
