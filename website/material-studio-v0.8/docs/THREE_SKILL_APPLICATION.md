# Three-Skill UI Redesign Application

## Anthropic `frontend-design`

Applied subject grounding, thesis-first hero design, typography as identity, meaningful structural devices, one justified signature, customer-side copy and anti-template self-critique.

Concrete result:
- replaced the prior dark luxury-dashboard vocabulary with a light architectural dossier;
- created the Surface Seam as the single signature;
- used “Complete surfaces, specified as one system” as the homepage thesis;
- replaced equal cards with a system index and changing architectural visual field;
- reserved the immersive dark treatment for Material Studio;
- removed public regression-test language, demo codes and engineering proof badges.

## UI UX Pro Max

Applied accessibility-first priority, 44px touch targets, responsive behavior, semantic tokens, readable typography, labelled forms, field feedback, WebP optimization, intrinsic dimensions, lazy loading, compatibility states and reduced motion.

Concrete result:
- semantic landmarks, skip links, one H1 and labelled controls;
- 16px body type and controlled line length;
- customer-facing concept imagery converted to WebP;
- mobile navigation and no-horizontal-overflow contracts;
- explicit material/profile compatibility and dimension availability;
- sample, quote, technical-support and distributor flows.

## Impeccable

The iteration represents the sequence:

`init → shape → critique → distill → typeset → layout → colorize → animate → adapt → harden → optimize → audit → polish`

Concrete result:
- `init`: PRODUCT.md and DESIGN.md;
- `shape`: light commercial site plus dark Studio;
- `critique`: baseline hierarchy, template, copy and wall defects;
- `distill`: removed dashboard panels and unnecessary proof UI;
- `typeset`: display/body/utility roles;
- `layout`: editorial ledgers and system indexes;
- `colorize`: mineral/copper/blueprint/evidence palette;
- `animate`: one hero reveal and material cross-fades;
- `adapt`: desktop, tablet and mobile rules;
- `harden`: explicit errors, consent, validation, rate limits, idempotency and security headers;
- `optimize`: WebP, dimensions, lazy loading and budgets;
- `audit`: deterministic anti-pattern and contract tests;
- `polish`: refined the Golden Wall from a visually framed overlay into the exact sloped replacement plane bounded by the ceiling and floor junctions.

## Resulting doctrine

The website is a commercial specification system, not a luxury dashboard and not a catalogue grid. The main site establishes understanding and trust; Material Studio provides one credible spatial decision; every commercial action terminates in a traceable route.
