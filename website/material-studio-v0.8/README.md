# Material Studio Commercial Website v0.8

This release re-audits and rebuilds the Material Studio commercial demo by combining three UI skills with distinct responsibilities:

1. `anthropics/skills` — `frontend-design`
2. `nextlevelbuilder/ui-ux-pro-max-skill`
3. `pbakaus/impeccable`

## Design result

- Light architectural-dossier commercial website rather than a generic black/gold configurator.
- One dark immersive Material Studio where the context justifies it.
- A single signature device, the **Surface Seam**, used as structure rather than decoration.
- Multi-page routes for Systems, Studio, Projects, Technology & Quality, Resources, About, Contact, Privacy and internal content status.
- Exact Golden Wall replacement plane following the real sloped ceiling and floor boundary.
- 24 material/profile overlays, compatibility rules, disabled-state reasons and public configuration references.
- Four traceable commercial intents: sample, quote, technical support and distributor.
- Evidence-backed public claims and controlled resources.
