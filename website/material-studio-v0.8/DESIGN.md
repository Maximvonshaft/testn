# Material Studio Design Context

## Physical scene
An architect reviews a material library in clear daylight, then enters a darker immersive room to compare a selected surface in context. The main website is precise, tactile and editorial; Material Studio is focused and cinematic.

## Point of view
Material Studio is not a luxury template and not an e-commerce catalogue. It is an architectural specification dossier with one memorable interactive seam between material evidence and spatial experience.

## Signature
**The Surface Seam** is a continuous copper reveal separating information from imagery in the hero, becoming the active system indicator and reappearing as the restrained construction reveal on the Golden Wall. It appears once per major composition, not on every card.

## Palette
- Mineral paper — primary light surface
- Chalk — high surface
- Mineral ink — primary text and dark Studio
- Umber — secondary dark surface
- Oxidized copper — action and seam
- Blueprint blue — technical references only
- Evidence green — approved evidence only

## Typography
- Display: Iowan Old Style / Palatino / Book Antiqua / Georgia
- Body: Aptos / Segoe UI / Helvetica Neue / sans-serif
- Utility: Cascadia Mono / SFMono / Consolas

Body type is at least 16px. Utility type is reserved for references, measurements and evidence status.

## Layout
- Editorial asymmetry instead of repeated equal cards.
- Hero is a thesis, not a metric dashboard.
- System navigation is an index with a changing visual field.
- Material Studio is one immersive dark interruption inside the light site.
- Trust information uses evidence rows and documents, not inflated statistics.

## Motion
- One orchestrated hero reveal.
- Cross-fade only for material changes.
- State transitions remain within 160–280ms.
- No bounce or parallax dependency.
- Reduced-motion removes nonessential movement.

## Anti-references
- Black-and-gold luxury dashboard templates
- Purple/blue SaaS gradients
- Identical icon-heading-text grids
- Tiny uppercase labels above every section
- Nested glass cards
- Giant metrics used as a hero substitute
