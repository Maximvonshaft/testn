# Resource Requirements

The current MVP is runnable without external image assets. A production version should replace the CSS-generated scene with real 2.5D assets.

## P0 required for production

1. Final English brand name and logo.
2. Final English product names.
3. Product dimensions by family: width, height, thickness, installation method and accessory requirements.
4. 5-10 real texture images: walnut slat, stone grey, oak, marble white, charcoal rib, outdoor decking.
5. Feature evidence: waterproof, low VOC / formaldehyde-free, fire resistance, scratch resistance, recycled content, installation method.
6. Aroma system data: aroma profiles, intensity, formulation duration wording, application limits.
7. Sample request routing and sales email.

## P1 recommended

1. Clean hero bedroom image without UI.
2. Mask files for wall, floor, furniture surface, trim and aroma zone.
3. Product sample box image.
4. Trim/accessory close-ups.
5. Installation process images.
6. Door-wall-cabinet same-color examples.
7. Factory/workshop trust images.

## P2 future

1. GLB models for sample box, wall panel cross-section, trim profiles and decking piece.
2. Lottie/Rive aroma animations.
3. Installation frame sequences or short transparent videos.
4. CAD/spec downloads.
5. Distributor policy content.
