# Material Configurator Spec

## Core state

```json
{
  "scene": "bedroom",
  "system": "Wall Panel",
  "pattern": "Walnut Linear",
  "texture": "Fine Ribbed",
  "finish": "Matte",
  "aroma": "Jasmine",
  "dimension": "Customized",
  "series": "Signature"
}
```

## Interactive zones

- Wall Surface
- Floor Surface
- Furniture Surface
- Trim System
- Aroma Layer

## View modes

### Design View

Default material and scene selection.

### Install View

Shows panel slide-in and trim finish logic.

### Layer View

Shows surface film, composite core, aroma layer and backing layer.

### Aroma View

Shows aroma atmosphere animation and aroma profile card.

## Configuration code

Generated from system, material, aroma and default dimension.

Example:

```text
MS-WPC-WALNUT-JASMINE-2400
```

## Output actions

- Request Sample
- Download Spec
- Ask for Quote
- Copy configuration code
