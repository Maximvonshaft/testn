# Material Studio Website Product Strategy

## Core positioning

The uploaded catalog shows a product matrix, not a single SKU: WPC wall panels, ASA decking, door panels, bamboo wood fiber panels, SPC wall panels, PVC marble/UV boards, ceiling panels, corners/trims, installation details and workshop proof.

The website should therefore avoid the generic position of "PVC/WPC panel manufacturer" and use:

> Composite Surface Systems for Premium Interiors.

## Product logic

Catalog SKUs become website systems:

- WPC wall panel -> Wall Surface System
- ASA decking -> Outdoor & Floor Surface System
- Door panel -> Door-Wall-Cabinet Matching System
- Bamboo wood fiber panel -> Interior Decorative Surface System
- SPC wall panel -> Wet Area Stone Surface System
- PVC marble/UV board -> Large Format Decorative Surface
- PVC ceiling panel -> Ceiling & Decorative Surface
- PVC corner/trim -> Finishing & Trim System

## First homepage logic

The first screen should be a space, not a catalog:

- a premium bedroom scene
- 5 hotspots only: Wall Surface, Floor Surface, Furniture Surface, Trim System, Aroma Layer
- material library at the bottom
- grouped console on the right
- configuration code and CTA

## Conversion logic

Every configuration must lead to:

- Request Sample
- Download Spec
- Ask for Quote
- Distributor inquiry
