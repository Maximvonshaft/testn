# Material Studio Website MVP v0.1

Static 2.5D homepage prototype for the composite building-materials brand.

## Product stance

This is not a traditional catalog website. The first website product is an interactive digital showroom:

> Composite Surface Systems for Premium Interiors.

## Product grounding

Based on the uploaded Zhongxin product catalog:

- WPC wall panels / slat panels
- ASA outdoor decking
- Door-wall-cabinet same-color system
- Bamboo wood fiber wall panels
- SPC stone wall panels
- 40/60cm wall panels
- PVC marble sheet / UV boards
- PVC ceiling panels
- PVC corner / trim / accessories
- Installation and workshop proof

## Run

Open `index.html` directly in Chrome/Edge.

For local serving:

```bash
python -m http.server 8080
```

Then open `http://127.0.0.1:8080`.

## MVP modules

- Centered navigation: Spaces / Materials / Studio / Technology / Projects / Distributor / Samples
- Scene selector: Bedroom / Bathroom / Hotel Suite / Retail / Outdoor
- 5 hotspots: Wall Surface / Floor Surface / Furniture Surface / Trim System / Aroma Layer
- Material Tuning Console
- Design / Install / Layer / Aroma modes
- Material Library
- Configuration code
- Request Sample / Download Spec / Ask for Quote CTAs
