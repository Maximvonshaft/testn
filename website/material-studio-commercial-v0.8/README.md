## Connector boundary

GitHub Connector is not a reliable transport for the 59 MiB generated scene assets and source catalogue. The complete runnable package is delivered separately. `source/material-studio-v0.8-text-bundle.json` contains the complete UTF-8 source tree, and `tools/materialize_v0_8_bundle.py` reconstructs it. Generated WebP/PNG assets and the catalogue still require direct Git, Git LFS, release-asset upload or object storage.

## Honest production boundary

Factory-approved SKUs, certifications, installed project evidence, production CRM/CMS credentials, legal privacy ownership and live domain/CDN/monitoring are external business dependencies and have not been fabricated.
