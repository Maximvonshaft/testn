# Material Studio Interactive Frontend v0.2

This is the first working frontend implementation based on the accepted Material Studio homepage concept.

## Run

Double-click `START_WINDOWS.bat`, then open:

```text
http://127.0.0.1:8080
```

Or:

```bash
python -m http.server 8080
```

## Working interactions

- choose Wall / Floor / Door-Wall / Ceiling / Trim / Aroma
- switch Design / Layer / Install / Aroma modes
- change pattern using catalog-derived texture crops
- change panel profile
- change tone, finish, aroma and dimension
- live configuration code generation
- save scheme to browser local storage
- copy code
- generate/download specification JSON
- sample request / quote dialog
- optional synthesized ambient sound

## Asset status

The downloadable v0.2 package includes:

- scene base derived from the approved concept mockup
- texture swatches derived from the Zhongxin catalog
- SVG surface masks
- profile icons
- aroma overlays
- scene thumbnails

This is a visual and interaction prototype. Production release still requires final product photography, validated technical claims, final brand identity, real enquiry routing and accessibility/performance QA.
