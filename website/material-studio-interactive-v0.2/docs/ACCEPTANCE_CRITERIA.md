# Acceptance Criteria

## Functional

- All six surface choices are selectable.
- Pattern, profile, tone, finish, aroma and dimension controls update state.
- Configuration code changes with selected parameters.
- Design, Layer, Install and Aroma modes are available.
- Scheme can be saved locally.
- Specification can be downloaded.
- Sample and quote dialogs open.
- Ambient sound is disabled by default and user-controlled.

## Experience

- No editor-only Add Layer, visibility, reorder or render controls.
- Product parameters are the primary interaction.
- The scene remains the main visual object.
- UI labels appear on hover/active state rather than permanently covering the space.
- Mobile fallback is usable even though the flagship experience is desktop-first.

## Quality

- JavaScript passes syntax checking.
- HTML parses without errors.
- All local asset paths resolve in a local HTTP server.
- No paid API or external CDN is required.

## Production blockers

- final brand identity
- final hero render/photography
- production masks
- verified product codes/dimensions
- verified technical and aroma claims
- form routing and privacy text
- accessibility and browser QA
